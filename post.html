Space Invader Game

This classic game was built using python library PyGame. Space Invaders is a fixed shooter in which the player moves a laser cannon horizontally across the bottom of the screen and fires at aliens overhead.

The goal is to eliminate all of the aliens by shooting them. While the player has one live, the game ends immediately if the invaders reach the bottom of the screen.

Technologies:
  • - Python
  • - PyGame
